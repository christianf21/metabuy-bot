How to Use


  1. Shoe Size: write down a shoe size to buy. Use dots instead of commas for decimals. For example: 10.5
  2. Keywords: write one keyword or more, separated by commas, that you know will appear in the tweet. Make sure it will actually appear.
  3. Twitter: write the twitter username (without the @) you want the bot to scan, normally it'll be @nikestore
  4. Options: you can select Automatic scan and it will check the twitter account every 1 second, if keywords match it will get the URL and open in new tab. If you have the link, select the Link option and open it using the Button. It will go and the bot will do its job.

Once the bot is in the nike.com website, it will automatically activate and select shoe size and add to cart depending on the settings you have set on the bot.

Before a release, make sure you have set the settings correctly. Test with your own twitter account using the Scan functionality and understand how the keywords work while scanning a tweet.
